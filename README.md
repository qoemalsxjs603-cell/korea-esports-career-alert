# 맞춤형 e스포츠 채용 공고 수집기 v12

v11 실제 diagnostic 결과를 바탕으로 감시 방식을 최종적으로 분리한 하이브리드 안정화판입니다.

## 감시 방식

- 10개 사이트: 공식 페이지 직접 수집
- 텐센트: 공식 채용 JSON API 사용
- KeSPA·한화생명e스포츠: JavaScript/접속 제한 때문에 공식 도메인 검색색인 RSS 보조 감시
- 디플러스 기아: 잡코리아의 현재 공식 회사 채용 페이지 사용

## 중요 안전장치

각 사이트는 처음 검증에 성공한 실행을 그 사이트만의 기준값으로 저장합니다.
따라서 현재 접속 불가인 사이트가 나중에 복구돼도 기존 공고가 한꺼번에 신규 메일로 오지 않습니다.

## 실행 순서

1. v12 파일로 완전히 교체
2. diagnostic 실행
3. last_run_report.json 확인
4. normal 한 번 실행
5. 이후 2시간마다 자동 실행

첫 normal 실행은 현재 공고를 기준값으로만 저장하며 메일을 보내지 않습니다.

KeSPA와 한화의 검색색인 방식은 공식 사이트 직접 수집보다 반영이 늦을 수 있지만,
오탐이나 대량 중복보다 안전성을 우선합니다.
